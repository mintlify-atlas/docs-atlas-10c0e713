## Overview

BMad Creative Intelligence Suite (CIS) brings structured creativity and innovation methodologies to the BMad Method ecosystem. It provides workflows and agents specialized in brainstorming, design thinking, creative problem-solving, and innovation facilitation.

**Module Code:** `cis`

## Key Features

- **Structured Brainstorming** - Facilitated ideation with proven techniques
- **Design Thinking Workflows** - Human-centered problem-solving processes
- **Innovation Frameworks** - Apply innovation methodologies systematically
- **Creative Problem-Solving** - Tackle complex challenges creatively
- **Idea Evaluation** - Assess and prioritize ideas objectively
- **Concept Development** - Transform ideas into actionable concepts
- **Creative Writing Support** - Content creation and storytelling
- **Innovation Agents** - Specialized personas for creative facilitation

## Installation

Install CIS during initial setup:

```bash
npx bmad-method install
# Select Creative Intelligence Suite when prompted
```

Or add it to an existing installation:

```bash
npx bmad-method install --modules cis
```

For non-interactive installation:

```bash
npx bmad-method install --modules cis --yes
```

### Design Thinking Process

Human-centered problem-solving:

```
/workflow design-thinking
```

**Phases:**
1. Empathize - Understand user needs
2. Define - Frame the problem
3. Ideate - Generate solutions
4. Prototype - Create tangible concepts
5. Test - Gather feedback and iterate

**Outputs:**
- Empathy maps
- Problem statements
- Solution concepts
- Prototype plans

## Best Practices

### Brainstorming

1. Set clear objectives and constraints
2. Defer judgment during ideation
3. Encourage wild ideas
4. Build on others' ideas
5. Go for quantity first, quality later

### Design Thinking

1. Maintain user empathy throughout
2. Embrace ambiguity and iteration
3. Prototype early and often
4. Gather diverse perspectives
5. Test assumptions with real users

### Idea Evaluation

1. Use objective criteria
2. Separate creation from judgment
3. Consider multiple dimensions
4. Involve diverse evaluators
5. Document decision rationale
