# BMad Method Roadmap

The BMad Method, BMad Method Module (BMM), and BMad Builder (BMB) are evolving rapidly. Here's what we're working on and what's coming next.

---

## In Progress

These features are actively being developed:

### 🧩 Universal Skills Architecture

**One skill, any platform. Write once, run everywhere.**

Create skills that work seamlessly across Claude Code, Cursor, Windsurf, and other AI IDEs without platform-specific rewrites.

### 🏗️ BMad Builder v1

**Craft production-ready AI agents and workflows.**

Comprehensive tooling for building custom agents with:
- Built-in evaluation systems
- Team collaboration features
- Graceful degradation handling
- Production-ready templates

### 🧠 Project Context System

**Your AI actually understands your project.**

Framework-aware context that evolves with your codebase:
- Automatic detection of frameworks and patterns
- Smart context gathering based on project type
- Dynamic updates as your project evolves

### 📦 Centralized Skills

**Install once, use everywhere.**

Share skills across projects without file duplication:
- Global skill repository
- Project-level overrides
- Version management
- Easy updates

### 🔄 Adaptive Skills

**Skills that know your tool.**

Optimized skill variants for different AI platforms:
- Claude-optimized workflows
- Codex-specific adaptations
- Kimi enhancements
- OpenCode integration
- And many more platforms

### 📝 BMad Team Pros Blog

**Guides, articles, and insights from the team.**

Launching soon with:
- Best practices and patterns
- Deep dives into workflows
- Case studies and examples
- Community spotlights
